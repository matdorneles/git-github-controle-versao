git init - para iniciar o git
git status - para verificar rastreamento dos arquivos
git add - para adicionar um arquivo ao rastreamento (. se for para a pasta toda)
git rm - para remover do rastreamento
git commit -m "mensagem" - para atualizar arquivo no repositório informando o que foi alterado
git log - mostra log, existem opções diferentes
git init --bare - inicia repositorio apenas para as alterações
git remote - lista repositorios remotos conhecidos
git remote add {url} {endereço} - cria repositorio remoto
git remove -v - mostra fetch e push dos endereços
git remote rename {nome-original} {novo-nome} - renomeia repositorio remoto
git clone {endereço} {nome da pasta para criar} - clona repositorio remoto
git push {nome-repositorio} {branch} - envia seus dados para o repositorio
git pull {nome-repositorio-remoto} {branch} - traz os dados de repositorio remoto
git branch {nome} - cria novo branch para trabalho
git checkout {branch} - seleciona qual branch trabalhar
git checkout -b {nome} - cria e vai para o branch
